## Installation

To get started with this project locally, follow these steps:

1. **Clone the repository:**

    ```bash
    git clone https://github.com/abbassimedayoub/holbertonschool-web_front_end.git
    ```

2. **Navigate to the project directory:**

    ```bash
    cd Bootstrap
    ```

3. **Install dependencies:**

    This project doesn’t require additional dependencies beyond Bootstrap itself, but ensure you have Bootstrap included in your HTML or linked via CDN.

4. **Open the project:**

    Open `index.html` in your web browser to see the project in action.

## Usage

1. **Include Bootstrap:**

    Make sure to include Bootstrap in your HTML file, either via CDN or by downloading it locally.

    **Using CDN:**

    ```html
    <link rel="stylesheet" href="https://stackpath.bootstrapcdn.com/bootstrap/4.5.2/css/bootstrap.min.css">
    <script src="https://code.jquery.com/jquery-3.5.1.slim.min.js"></script>
    <script src="https://cdn.jsdelivr.net/npm/@popperjs/core@2.5.3/dist/umd/popper.min.js"></script>
    <script src="https://stackpath.bootstrapcdn.com/bootstrap/4.5.2/js/bootstrap.min.js"></script>
    ```

    **Or download Bootstrap and include it locally:**

    ```html
    <link rel="stylesheet" href="path/to/bootstrap.min.css">
    <script src="path/to/jquery.min.js"></script>
    <script src="path/to/bootstrap.min.js"></script>
    ```

2. **Customize Bootstrap:**

    Modify `custom.css` to add your own styles and override Bootstrap defaults as needed.

3. **Develop:**

    Edit `index.html` and other files to build out your site's layout and functionality.
