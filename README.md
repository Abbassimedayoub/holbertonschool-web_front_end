# Web Front End — HTML, CSS & JavaScript Track

## Overview

Holberton School front-end track: **12 projects** going from semantic HTML to advanced CSS, Sass, accessibility, responsive design and JavaScript in the browser. Each directory is an independent project with its own tasks.

## Modules

| Module | Focus |
|---|---|
| `html_advanced/` | Semantic HTML5 structure of a full page |
| `CSS_advanced/` | Selectors, custom properties, typography, layout |
| `flexbox/` | Flexbox-based layouts |
| `responsive_design/` | Media queries, fluid layouts, mobile-first |
| `accessibility/` | WCAG conformance, ARIA, keyboard navigation, skip links |
| `form/` | Accessible, styled and validated HTML forms |
| `sass_scss/` | Variables, nesting, mixins, inheritance |
| `Bootstrap/` | Grid and components with Bootstrap |
| `developer_tools/` | Browser DevTools: audit, network, performance |
| `Javascript_advanced/` | Closures, scoping, callbacks, binding |
| `JQuery_advanced/` | DOM manipulation, events, AJAX with jQuery |
| `Cookies_local_storage/` | Client-side state: cookies and Web Storage |

## Technologies

HTML5, CSS3, Sass/SCSS, Bootstrap, JavaScript (ES6), jQuery.

## Skills Demonstrated

Semantic and accessible markup, scalable CSS architecture, responsive layouts, browser tooling, and client-side JavaScript patterns — the foundations under any front-end framework.
